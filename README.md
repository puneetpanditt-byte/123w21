# SDGD College Website

## Static HTML Version for GitHub Pages

This is the static HTML version of the SDGD College website, converted from PHP to work on GitHub Pages.

### Features
- Responsive design
- Modern UI with Bootstrap-style components
- College information and courses
- Photo and video galleries
- Contact information

### Deployment

- JavaScript for interactivity
- Font Awesome for icons
- Responsive design principles
